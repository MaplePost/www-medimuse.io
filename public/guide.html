Empatica API Key Instructions

On your browser, go to: http://www.empatica.com/connect/login.php   Log in using your email & password.  If you have not set up an E4 Connect account do so first with the Create Account link.

Make sure you are on the Home Page. The click on the Developer link

On the Developer page, find API Key; a 32 alpha-numeric sequence in the following format: 0000000aa0000000aaaa0000000a000a. Select & copy.  Paste this key in the API Key entry window in the app.  You will also want to add your device on this page to link your E4 Device with the API Key

In the Statesong app the API Key dialog will come up on startup if you have not registered your key, you can also access this the API Key Dialog using the Devices menu item at any time.

In the Statesong app API Key dialog screen paste your API Key
