template<class T>
class IInputTrack< T >

Public interface for an input track of biometric data.

Author
Peter J Slack (pslack)An Input track is a preset of data ticks organized into an array. The timing of how these data points are input to the model via the Input signal member are controlled by the model audio stream timing (framed_time_tick).
