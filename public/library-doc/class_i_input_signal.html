Detailed Description

template<class T>
class IInputSignal< T >

Public interface for a real time biometric input signal.

Author
Peter J Slack (pslack)Input signals are real time points from where to send data to the MapleModel. Input signals are tagged with an enumerated type so as to sort them appropriately for processing. Input signals are meant to be real time however they are assuming that a data_tick happens at regular frequency ie. this is a discrete data input channel as opposed to a random input data set. It is up to the user to send data at the regular interval.
